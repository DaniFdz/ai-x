# ai-x

A specialized agent for scraping and summarizing the latest AI news from Twitter/X.

## Purpose

Engineers following AI-related accounts on Twitter/X are overwhelmed by the constant stream of information. This agent helps them stay up-to-date without doomscrolling for hours by:

- Scraping tweets from key AI accounts and topics
- Filtering and categorizing relevant AI news
- Summarizing the most important updates into digestible digests
